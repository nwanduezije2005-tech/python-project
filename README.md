# SmartPlate

SmartPlate is a Python-based system that automates plate tracking and billing in conveyor belt sushi restaurants.

## Features
- Tracks plates in real time
- Calculates total cost automatically
- Simple Flask web interface

## Technologies Used
- Python
- Flask

## How to Run
1. Install Flask
2. Run the Python file:
   python p2m5ezijenwandu.py
3. Open browser:
   http://127.0.0.1:5000
