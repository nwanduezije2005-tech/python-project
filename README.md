# SmartPlate

SmartPlate is a Python-based system designed for conveyor belt sushi restaurants. The system helps automate plate tracking, customer billing, and food ordering through a simple Flask web application.

## Features
- Tracks sushi plates in real time
- Calculates customer totals automatically
- Simulates restaurant billing logic
- Uses a Flask-based web interface

## Technologies Used
- Python
- Flask
- GitHub

## Project Goal
The goal of this project was to explore how Python and Flask can be used to create a restaurant automation system that improves efficiency and customer experience.
